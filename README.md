# how to run the project

To run this project, follow the steps below:
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/your-repository.git
    cd your-repository
    ```
2. **Install dependencies**:
  ```bash
  pip install -r requirements.txt
  ```
3. **Run the project**:
   ```bash
   python scraper.py
   ```
