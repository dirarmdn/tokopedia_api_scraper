To run this project, follow the steps below:
1. **Clone the repository**:
    ```bash
    https://github.com/dirarmdn/tokopedia_api_scraper.git
    cd tokopedia_api_scraper
    ```
2. **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3. **Run the project**:
    ```bash
    python scraper.py
    ```
